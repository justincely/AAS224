Splittag

Splittag is a python module to split a COS corrtag file apart in time. The created files can then be re-extracted using calcos or x1dcorr.

In PyRAF, this can also be found in the stsdas.hst_calib.hstcos package as splittag.

This notebook will provide a complete walkthrough to using this module in a common task.
